# **App Name**: Prime Metric

## Core Features:

- Calculator Tool: Provides access to a variety of calculators.
- Educational Tool: Offer educational tools such as glossaries, guides and tutorials.
- Animated Hero Section: Creates a captivating animated section with floating elements to instantly grab user attention.
- Glass Effect Navigation Bar: Allows users to navigate with a glass effect.
- Sticky Search Bar: Integrate a sticky glass search bar for quick and easy access.
- FAQ Section: Display the frequently asked questions.
- Content summarizer: Summarizes large pieces of content from articles and tutorials to highlight the key information, with a tool determining the degree of simplification.

## Style Guidelines:

- Primary color: Deep futuristic blue (#1E3A8A) to convey a tech-forward and trustworthy feel.
- Background color: Dark, desaturated blue (#212A3D) for a modern dark-glass aesthetic.
- Accent color: Electric purple (#A21CAF) for gradient CTA buttons, highlights and accents.
- Body text: 'Inter' sans-serif font.
- Headline text: 'Merriweather' serif font.
- Use crisp, minimal vector icons, and floating UI elements.
- Mobile-first layout with clear category sections, and tool cards. SEO-friendly, and AdSense-compliant.
- Soft, smooth, and minimal animations with fade-slide transitions, parallax effects, and floating micro-animations.
